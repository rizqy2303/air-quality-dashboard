# Air Quality Analysis Dashboard: Beijing Dataset

Ini adalah proyek akhir (Submission) untuk kelas **Belajar Analisis Data dengan Python** di platform Dicoding. Proyek ini bertujuan untuk menganalisis tren polusi PM2.5 di Beijing dan menyajikannya dalam dashboard interaktif.

## Struktur Direktori
Pastikan struktur folder Anda adalah sebagai berikut agar dashboard berjalan lancar:
```text
.
├── dashboard/
│   ├── dashboard.py       # File utama aplikasi Streamlit
│   └── main_data.csv      # Dataset yang telah dibersihkan (Cleaned)
├── data/
│   └── (File CSV mentah dari stasiun pemantau)
├── notebook.ipynb         # Proses analisis data lengkap
├── README.md              # Dokumentasi proyek
├── requirements.txt       # Daftar library yang dibutuhkan
└── url.txt                # Link dashboard online
